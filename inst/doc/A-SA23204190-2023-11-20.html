Question1

为什么使用unlist()把表转化为原子向量，为什么不用as.vector()？

Answer1

unlist()将表转化为一个向量，as.vector()可能会将表格分为多个向量。

u<-list(1,2,3,4)
x<-unlist(u)
y<-as.vector(u)
x

y
## [[1]]
## [1] 1
## 
## [[2]]
## [1] 2
## 
## [[3]]
## [1] 3
## 
## [[4]]
## [1] 4

Question6

下面是一个数据缩放函数，如何将其应用到数据框的每一列，如何将其应用到数据框的每一数字列

scale01 <- function(x) { rng <- range(x, na.rm = TRUE) (x - rng[1]) / (rng[2] - rng[1]) }

Answer6

scale01 <- function(x) {
  rng <- range(x, na.rm = TRUE)
  (x - rng[1]) / (rng[2] - rng[1])
}

Question8

现有二元密度函数 \[f(x,y)\propto \begin{pmatrix} n \\ x \end{pmatrix}y^{x+\alpha-1}(1-y)^{n-x+b-1},\ x=0,1,...,n,0\leq y\leq 1 \] 可以看出，对于固定的a,b,n,条件分布为Binomial(n,y)和Beta(x+a,n-x+b)，使用Gibbs样本生成器生成一个目标联合密度为f(x,y)的链条。

Answer8

N<-5000
burn<-1000
X<-matrix(0,N,2)
a<-b<-3;n<-100#固定a=3,b=3,n=100


X[1,]<-c(1,0.5)
for (i in 2:N) {
  x2<-X[i-1,2]
  X[i,1]<-rbinom(1,n,x2)
  x1<-X[i,1]
  X[i,2]<-rbeta(1,x1+a,n-x1+b)
}

b<-burn+1
x<-X[b:N,]



plot(x,main = "",cex=.5,xlab = bquote(X[1]),
     ylab = bquote(X[2]),ylim = range(x[,2]))
